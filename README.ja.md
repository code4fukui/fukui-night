# 厳選 ふくいの夜マップ

福井県福井市の夜の街を厳選した地図アプリです。福井経済同友会によって作成されました。

## デモ
https://code4fukui.github.io/fukui-night/

地図上に店舗情報が表示されており、時間指定で営業中の店舗を絞り込めます。

## 機能
- 福井市内の飲食店情報を地図上に表示
- 時間指定で営業中の店舗を絞り込み表示
- 店舗情報の一覧表示

## 必要環境
特にありません。ウェブブラウザで利用可能です。

## 使い方
特にインストールは不要で、上記のデモページにアクセスするだけで利用できます。

## データ
店舗情報は「厳選 ふくいの夜」のウェブサイトから収集したオープンデータを使用しています。
