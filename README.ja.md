# fukui-night

福井経済同友会のデータに基づき、福井市内の夜のスポットを厳選してまとめたインタラクティブなマップです。

## デモ

https://code4fukui.github.io/fukui-night/

このアプリケーションは、レストランやバーなどの施設を表示するインタラクティブなマップと、フィルタリング可能なテーブルを備えています。

## 機能

- **インタラクティブなマップ**: 福井市の主要エリア（福井駅前、片町など）の夜のスポットを表示します。
- **時間によるフィルタリング**: 指定した時間、または「今」営業している店舗のみを表示します。
- **柔軟な時間解析**: 日をまたぐ営業時間（例: `19:00-02:00`）やラストオーダー（`L.O.`）など、複雑な時間表記にも対応しています。
- **詳細な店舗情報**: 各スポットのカテゴリ、住所、営業時間、価格帯、席数、Google Mapsへの直接リンクを確認できます。
- **テーブルビュー**: マップの下部には、全店舗を対象としたフィルタリングおよびソート可能なテーブルが用意されています。

## 技術概要

このプロジェクトは、標準的なHTML、CSS、JavaScriptモジュールで構築された静的Webアプリケーションです。バックエンドは必要ありません。

### データパイプライン

データは [fukui-night.com](https://fukui-night.com/) から取得しています。Denoスクリプトを使用した2段階のパイプラインで生成・処理されます。

1. **`download.js`**: ソースサイトから店舗の詳細（店名、住所、営業時間など）をスクレイピングし、`fukui-night.csv` に保存します。
2. **`addlatlng.js`**: 中間CSVを読み込み、`getLatLngFromGMap.js` を使用してGoogle MapsのURLから緯度・経度を解決し、アプリケーションで使用する最終的な `fukui-night_pos.csv` を出力します。

データを再生成するには、以下のコマンドを実行します:
```bash
deno run -A download.js
deno run -A addlatlng.js
```
