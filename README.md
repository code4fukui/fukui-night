# fukui-night

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Curated map of night spots in Fukui, Japan, created by the Fukui Business Society.

## Demo
https://code4fukui.github.io/fukui-night/

## Features
- Interactive map of restaurants, bars, and other night spots in Fukui
- Filters to show only open establishments at a given time
- Detailed information on each location including category, name, address, hours, pricing, and more

## Requirements
None, the project is hosted online.

## Usage
The map and information can be accessed directly through the demo link.

## Data
The data for this project is scraped from the [fukui-night.com](https://fukui-night.com/) website.
